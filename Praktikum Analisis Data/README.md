## 🧠 Tentang Proyek
Proyek **Analisis Nilai Siswa** ini dibuat menggunakan **Python** untuk membantu menganalisis data nilai siswa dari file CSV.  
Program ini bisa:
- Menghitung **rata-rata nilai** semua siswa  
- Menentukan **nilai tertinggi dan terendah**  
- Menampilkan **laporan hasil analisis** dengan cara cepat dan efisien  

💬 Proyek ini cocok untuk latihan dasar analisis data dan logika pemrograman Python.
