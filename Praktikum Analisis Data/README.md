## 🧩 Tentang Proyek
**Analisis Nilai Siswa** adalah proyek Python yang digunakan untuk **menganalisis data nilai siswa dari file CSV**.  
Program ini membantu menampilkan **rata-rata nilai**, **nilai tertinggi**, dan **nilai terendah** dari sekumpulan data dengan cepat dan efisien.

🎯 Tujuan utamanya adalah untuk mempelajari dasar **analisis data** menggunakan **Python**, serta memperkuat pemahaman logika pemrograman yang bermanfaat untuk dunia nyata.

## ⚙️ Fitur Utama
✨ Membaca data dari file **CSV** secara otomatis  
📊 Menghitung nilai **rata-rata, tertinggi, dan terendah**  
📈 Menampilkan hasil analisis dengan format yang mudah dibaca  
💡 Dapat dikembangkan dengan library **Pandas** untuk analisis lanjutan

## 🚀 Cara Menjalankan Program
1. Pastikan sudah terpasang **Python 3.x** di perangkatmu.  
2. Simpan file berikut di folder yang sama:
   - `analisis_nilai.py`
   - `nilai_siswa.csv`
3. Buka terminal / command prompt, lalu jalankan:
   ```bash
   python analisis_nilai.py
